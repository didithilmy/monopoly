# Engi's Monopoly
A mock of one of the world's most popular board game: Monopoly.

### Build and Execution

Building this project requires Maven build tool and Java 8 (or newer).

```sh
$ mvn install
$ java -jar ./target/monopoly-1.0.jar
```
